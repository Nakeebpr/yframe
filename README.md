# Colorful Todo App

A beautiful, colorful todo application built with React.js.

## Features

- ✨ Add new todos
- ✅ Mark todos as complete
- 🗑️ Delete todos
- 🎨 Each todo gets a random colorful border
- 📊 View statistics (total, completed, remaining)
- 🎯 Modern and responsive UI

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- Vite
- CSS3 (with gradients and animations)

Enjoy your colorful todo app! 🎉
